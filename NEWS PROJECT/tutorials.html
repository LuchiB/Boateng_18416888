LINKING CSS

CSS can be added to HTML by linking to a separate stylesheet file, importing files from existing stylesheets, embedding CSS in a style tag, or adding inline styles directly to HTML elements. Many of these methods can also be done with javascript.

This is the most common method of attaching CSS rules to HTML documents. With this method, all your style rules are contained in a single text file that's saved with the .css extension. This file is saved on your server and you link to it directly from each HTML file. We use a link tag which is a simple line of HTML that you put in the head section of your HTML document
